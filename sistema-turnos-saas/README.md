## 📅 Planificador de Tareas con Calendario Interactivo 

Aplicación web que permite organizar tareas diarias vinculadas a fechas específicas mediante un calendario interactivo. 
El usuario puede agregar, marcar como completadas y eliminar tareas. Los datos se almacenan en el navegador utilizando LocalStorage, 
lo que permite mantener la información incluso al recargar la página. 

--- 

## 🛠 Tecnologías utilizadas 

- HTML5
- JavaScript (Vanilla JS)
- LocalStorage
- CSS3

--- 

## ✨ Funcionalidades 

- Selección de fecha desde calendario interactivo
- Agregar nuevas tareas - Marcar tareas como completadas
- Eliminar tareas
- Persistencia de datos con LocalStorage
- Diseño responsive

--- 

## 🎯 Objetivo del proyecto 

Proyecto desarrollado como práctica de JavaScript para reforzar conceptos de: 
- Manipulación del DOM
- Eventos
- Almacenamiento en el navegador
- Organización de datos
